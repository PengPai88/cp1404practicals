# CP1404 Practical 02 Reflection

## What did you learn?
I learned how to use functions to break down code into smaller, reusable parts. I also practiced using Git and GitHub to version control my work.

## What was challenging?
The hardest part was remembering not to print inside functions — I kept wanting to output results directly instead of returning them.

## How did using functions help?
Using functions made my code cleaner and easier to test. For example, I could reuse the same `determine_result()` function for both user input and random scores.

## Did you use Git correctly?
Yes, I committed each file with meaningful messages and pushed to GitHub at the end. I now understand that pushing regularly means I won’t lose my work.

## Any other thoughts?
It’s helpful to write code on paper first — it helps me think about structure before typing.
